## 프로젝트 소개
* 폭탄을 피해 높은 점수를 얻는 간단한 게임
* 폭탄을 오래 피할수록 점수가 올라가고, 스테이지는 1~7단계로 이루어지며 150점 점수가 증가할 때마다 스테이지 변경
* 폭탄은 랜덤으로 터지며, 터진 불꽃에 닿으면 플레이어는 사망
* 높은 스테이지일 수록 터지는 시간 감소, 불꽃 지속 시간 증가
* 5위 안에 들었을 때 플레이어 이름을 입력하면 1~5위까지 순위 랭킹 갱신
